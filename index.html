VNlytics

Analytics for story based games.

Download .zip Download .tar.gz View on GitHub

VNlytics is a narrative analytics system for story based games.

It enables player choices to be recorded for developer analysis, or additionally returned to the player, to compare their choices with others.

Usage

The system was designed for the Ren'Py Visual Novel Engine, but should be easily reconfigurable to work with other projects.
